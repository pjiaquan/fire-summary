Settings

Fire Summary

管理 Gemini API、目標語言、Prompt 與 shortcut 偏好。

API Parameters

控制 Gemini 生成參數。搜尋功能只會在支援的模型上生效。

Shortcut

桌面版瀏覽器預設快捷鍵是 `Alt+X`，會直接開啟 Fire Summary popup 並自動開始摘要。

Android Firefox 不支援 extension commands，請從擴充功能選單開啟 Fire Summary。

如果要修改桌面版快捷鍵，請到瀏覽器的 extension shortcuts 設定頁調整。

Typography

控制 popup 摘要與延伸討論頁的標題字體、內文字體、字重與行高。

Cache Protection

只快取 AI 摘要結果，不保存全文。快取會自動套用有效期與筆數上限，超出會自動清理。
